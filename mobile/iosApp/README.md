# iOS App Modülü

Bu klasör iOS uygulaması için kullanılacaktır.

## Oluşturulacak Yapı

```
iosApp/
├── Evalon/
│   ├── App.swift
│   ├── ComposeView.swift
│   ├── Info.plist
│   └── Assets.xcassets
├── Evalon.xcodeproj
└── Evalon.xcworkspace
```

## Kurulum (macOS'ta)

1. Xcode'da yeni iOS App projesi oluştur
2. Shared framework'ü entegre et
3. ComposeView.swift ekle
4. Build ve run!

Detaylar için `IOS_PLAN.md` dosyasına bakın.
